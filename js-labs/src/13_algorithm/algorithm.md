## Pseudo code - Mã giả

```js
FUNCTION findMax
INPUT:
  numberList - an array of numbers
OUTPUT:
  the maximum number
---
IF (numberList is not an array or array is empty)
  RETURN undefined

SET max = first element
FOR each number in numberList
  IF current number > max
  THEN SET max = current number

RETURN max
```
